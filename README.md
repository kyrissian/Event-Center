## 🏠 Introduction

**SoCal BJJ Event Center** is a multi-page, responsive website designed to showcase a Brazilian Jiu-Jitsu event venue based in Southern California. Built as a front-end web development project, the site simulates a real-world event center online presence — featuring upcoming tournaments, past event galleries, a venue snack menu, location info, and a contact form for inquiries.

The site is designed with a sleek dark theme, bold typography, and a blue accent color palette that gives it a professional, sports-arena feel. It is fully responsive across desktop, tablet, and mobile screen sizes.

This project demonstrates mastery of **semantic HTML5**, **external CSS3 styling**, **CSS Grid & Flexbox layouts**, **responsive design**, and **interactive UI elements**.

---

## ✨ Features

- **Multi-page site** with consistent navigation across all pages
- **Responsive hamburger menu** for mobile and tablet viewports
- **Hero banner** with hover shadow effect on the home page
- **Event cards** organized by season (Spring, Summer, upcoming) with ticket purchase buttons
- **Past events gallery** using CSS Grid with a "Back to Top" anchor link
- **Snack menu page** with a styled HTML table and frosted glass header effect
- **Contact form** with name, email, and message fields
- **Embedded Google Map** on the location page
- **Social media footer links** with hover animations (Facebook, Twitter, Instagram, LinkedIn)
- **Custom SVG favicon** (BJJ fist icon) across all pages
- **Font Awesome icons** used throughout for visual flair
- **Hover & focus effects** on buttons, nav links, and the hero section
- **CSS Grid** for the event and gallery card layouts
- **Flexbox** for header, nav, footer, and form layouts
- **Dark color scheme** (`#0f172a` background) with blue accents (`#4259bd`)

## 🛠 Prerequisites

No build tools or dependencies are required. The project runs entirely in the browser using:

- A modern web browser (Chrome, Firefox, Safari, Edge)
- An internet connection (for Font Awesome CDN icons and any external images)

---

## 💻 Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/socal-bjj-event-center.git
   ```

2. **Navigate into the project folder:**
   ```bash
   cd socal-bjj-event-center
   ```

3. **Open the site in your browser:**
   - Simply open `index.html` directly in your browser, **or**
   - Use the [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) extension in VS Code for a local dev server

---

## 🚀 Usage

Once opened in a browser, you can navigate the full site using the top navigation bar (or hamburger menu on mobile):

| Page | File | Description |
|------|------|-------------|
| Home | `index.html` | Hero banner + spotlight upcoming events |
| Events | `events.html` | Full seasonal event listings with ticket buttons |
| Menu | `menu.html` | Venue concession snack menu table |
| Location | `location.html` | Embedded map + venue address and details |
| Gallery | `gallery.html` | Photo grid of past BJJ events |
| Contact | `contact.html` | Inquiry form with name, email, and message |

All pages share a single external stylesheet (`newstyles.css`) and are fully responsive at three breakpoints: desktop (1000px+), tablet (768px), and mobile (599px and below).
